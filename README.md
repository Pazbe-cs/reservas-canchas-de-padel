# Sistema de Reservas de Canchas de Pádel

La idea es construir una aplicación que permita gestionar reservas de canchas de pádel utilizando:

- Backend: Java (Quarkus)
- Frontend: Angular
- Control de versiones: Git y GitHub
- Contenedores: Docker

## Descripción del proyecto

El sistema permitirá consultar canchas disponibles, ver horarios, registrar reservas y evitar superposición de turnos.  
El backend se encargará de la lógica del sistema y el frontend será la parte visual donde interactúa el usuario.
